1. Installation

Alkhalil morpho sys is written in Java. So to use the program, you must install Java Virtual Machine version 1.4 or later. This means that the program can be used in several systems such as Windows, Linux, Solaris, MacOs. The virtual machine can be downloaded from the Sun Microsystems web site at the following address: http://java.sun.com
